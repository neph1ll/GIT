1. Скопировать репозиторий с GitHub на свой компьютер:

Предварительные шаги:

Установить Git на свой компьютер.
Зарегистрироваться на GitHub, если вы еще не зарегистрированы.
Создать SSH-ключ для безопасного взаимодействия с GitHub.
Копирование репозитория:

Скопировать URL репозитория с GitHub.
Открыть терминал (командную строку) на своем компьютере.
Перейти в каталог, где вы хотите сохранить репозиторий.
Выполнить команду: git clone <URL репозитория>

Вставить URL репозитория, который вы скопировали.

2. Создать Git репозиторий и отправить на GitHub:

Создание репозитория:

Открыть терминал в локальном каталоге проекта.
Выполнить команды:
git init
git add .
git commit -m "First commit"

Отправка на GitHub:

Создать новый репозиторий на GitHub (если его еще нет).
Скопировать URL репозитория.
Выполнить команду:
git remote add origin <URL репозитория>
git push -u origin master

3. Получить изменения с GitHub на локальный компьютер:

Выполнить команду:
git pull origin master


4. Информация о состоянии репозитория:

git status: Показывает измененные, добавленные и неотслеженные файлы в вашем репозитории.

git log: Показывает историю коммитов в репозитории.

git remote -v: Показывает URL-адреса удаленных репозиториев.
